# vue-cli搭建的vue项目更改端口

vue-cli 3.0及以上

// 在根目录下创建 vue.config.js 文件,填入以下内容：
module.exports = {
    devServer: {
        port: 8080,     // 端口
        open: true      // 启动后是否自动打开网页
    },
    lintOnSave: false   // 取消 eslint 验证
};
